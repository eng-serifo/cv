W3C: WebAuthn é agora um padrão oficial da Web

Serifo Baldé

WebAuthn recém-aprovado permite que você faça login em todas as suas contas sem uma senha.

Diga adeus às senhas e passe para a autenticação na Web, pois o World Wide Web Consortium (W3C) e a FIDO Alliance anunciaram na segunda-feira a finalização de um padrão da Web para logins seguros e sem senha. Foi agora aprovado como padrão oficial da web. "Esse avanço é um grande passo para tornar a web mais segura - e utilizável - para usuários em todo o mundo", disse o W3C em seu comunicado à imprensa . A especificação de Autenticação da Web (WebAuthn) permite que os usuários façam login em suas contas da Internet usando seu dispositivo preferido sem precisar se lembrar de senhas. Em vez disso, os usuários podem fazer login usando dados biométricos, como impressões digitais, chaves de segurança USB ou dispositivos como smartphones ou relógios.

O W3C afirma que isso tornará os sites mais seguros e dará mais segurança às senhas. Ele já é suportado nos navegadores da Web do Windows 10, Android e Google Chrome, Mozilla Firefox, Microsoft Edge e Apple Safari (visualização). O W3C recomenda que os sites adotem o novo padrão para criar um ambiente mais seguro para os usuários e permitir que eles efetuem login com mais facilidade, rapidez e segurança: "Agora é a hora de os serviços e empresas da Web adotarem o WebAuthn para ir além das senhas vulneráveis ​​e ajudar os usuários da Web a melhorar a segurança de suas experiências on-line", disse Jeff Jaffe, CEO do W3C. “A Recomendação do W3C estabelece uma orientação de interoperabilidade para toda a web, estabelecendo expectativas consistentes para os usuários da web e os sites que eles visitam. O W3C está trabalhando para implementar essa melhor prática em seu próprio site. ”

Além disso, o W3C considera que o WebAuthn irá erradicar muitos problemas relacionados aos métodos tradicionais de autenticação. “É de conhecimento comum que as senhas ultrapassaram sua eficácia. Não são apenas senhas roubadas, fracas ou padrão atrás de 81% das violações de dados, elas são um desperdício de tempo e recursos. Embora as soluções tradicionais de autenticação multifatoriais (MFA), como códigos únicos de SMS, adicionem outra camada de segurança, elas ainda são vulneráveis ​​a ataques de phishing, não são simples de usar e sofrem com baixas taxas de adesão. Com a WebAuthn, a comunidade global de tecnologia se uniu para fornecer uma solução compartilhada para o problema de senha compartilhada ”.

Por outro lado, as chaves FIDO têm muitas vantagens sobre senhas e abordam todos os problemas com autenticação tradicional, como:

Segurança : As credenciais de login criptográfico do FIDO2 são únicas em todos os sites, biometria ou outros segredos, como senhas, nunca deixam o dispositivo do usuário e nunca são armazenadas em um servidor. Esse modelo de segurança elimina os riscos de phishing, todas as formas de roubo de senhas e ataques de repetição.

Conveniência : os usuários fazem login com métodos convenientes, como leitores de impressão digital, câmeras, chaves de segurança FIDO ou seu dispositivo móvel pessoal.

Privacidade : Como as chaves FIDO são exclusivas para cada site da Internet, elas não podem ser usadas para rastreá-lo nos sites.

Escalabilidade : os sites da Web podem ativar o FIDO2 por meio de chamadas de API simples em todos os navegadores e plataformas suportados em bilhões de dispositivos que os consumidores usam todos os dias.

"A autenticação na Web como padrão oficial da Web é o ápice de muitos anos de colaboração no setor para desenvolver uma solução prática para autenticação mais forte na Web", disse Brett McDowell, diretor executivo da FIDO Alliance. "Com este marco, estamos nos movendo para uma nova era de proteção de autenticação FIDO onipresente e suportada por hardware para todos que usam a Internet". O anúncio da WebAuthn do W3C e da FIDO Alliance é, esperamos, um passo no sentido de obter logins sem senha em sites. Esperamos ver muitos serviços da Web implementando o WebAuthn nos próximos meses e, eventualmente, um uso mais amplo dele na Web como um todo.
