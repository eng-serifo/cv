W3C : WebAuthn est désormais un standard web officiel

Serifo Baldé

WebAuthn nouvellement approuvé vous permet de vous connecter à tous vos comptes sans mot de passe.

Dites adieu aux mots de passe et passez à l'authentification Web en tant que World Wide Consortium Web (W3C) et l'Alliance FIDO a annoncé lundi la finalisation d'un standard web pour les logins sécurisé et sans mot de passe. Il a maintenant été approuvé en tant que norme Web officielle. "Cette avancée est un grand pas en avant pour rendre le Web plus sûr - et plus utilisable - pour les utilisateurs de dans le monde entier », a déclaré le W3C dans son communiqué de presse. La spécification d'authentification Web (WebAuthn) permet aux utilisateurs de se connecter à leur Comptes Internet utilisant votre appareil préféré sans avoir à mémoriser les mots de passe. À la place De plus, les utilisateurs peuvent se connecter à l'aide de données biométriques telles que les empreintes digitales, les clés Périphériques de sécurité USB ou périphériques tels que les smartphones ou les montres.

Le W3C affirme que cela rendra les sites Web plus sûrs et les mots de passe plus sûrs. il est déjà pris en charge sur les navigateurs Web Windows 10, Android et Google Chrome, Mozilla Firefox, Microsoft Edge et Apple Safari (préversion). Le W3C recommande que les sites Web adoptent la nouvelle norme pour créer un environnement plus sûr pour utilisateurs et leur permettre de se connecter plus facilement, rapidement et en toute sécurité : "Il est maintenant temps pour les services Web et les entreprises d'adopter WebAuthn pour aller au-delà des mots de passe vulnérables et aider les internautes à améliorer la sécurité de leurs expériences en ligne », a déclaré Jeff Jaffe, PDG du W3C. « La recommandation du W3C énonce une ligne directrice pour interopérabilité pour l'ensemble du Web, définissant des attentes cohérentes pour les utilisateurs du Web et les sites qu'ils visitent. Le W3C travaille à la mise en œuvre de cette meilleure pratique dans votre propre site web. ”

De plus, le W3C estime que WebAuthn éliminera de nombreux problèmes liés à méthodes d'authentification traditionnelles. « Il est de notoriété publique que les mots de passe ont dépassé leur efficacité. Ce ne sont pas que des mots de passe volés, faibles ou par défaut à l'origine de 81% des violations de données, ils sont une perte de temps et ressources. Alors que les solutions traditionnelles d'authentification multifacteur (MFA), telles que Les messages SMS, ajoutent une autre couche de sécurité, ils sont toujours vulnérables aux attaques par e-mail. phishing, ne sont pas simples à utiliser et souffrent de faibles taux d'adhésion. Avec WebAuthn, le la communauté technologique mondiale s'est réunie pour fournir une solution commune au problème mot de passe partagé”.

D'autre part, les clés FIDO présentent de nombreux avantages par rapport aux mots de passe et résolvent tous les problèmes avec une authentification traditionnelle, telle que:

Sécurité : les identifiants de connexion cryptographiques FIDO2 sont uniques les sites Web, la biométrie ou d'autres secrets tels que les mots de passe ne quittent jamais le utilisateur et ne sont jamais stockées sur un serveur. Ce modèle de sécurité élimine les risques de phishing, toutes formes de vol de mot de passe et les attaques par rejeu.

Commodité : les utilisateurs se connectent avec des méthodes pratiques telles que empreintes digitales, caméras, clés de sécurité FIDO ou votre appareil mobile personnel.

Confidentialité : Les clés FIDO étant propres à chaque site internet, elles ne sont pas peut être utilisé pour vous suivre sur des sites Web.

Évolutivité : les sites Web peuvent activer FIDO2 via de simples appels d'API sur tous les navigateurs et plates-formes pris en charge sur des milliards d'appareils qui consommateurs utiliser tous les jours.

"L'authentification Web en tant que norme Web officielle est l'aboutissement de nombreuses années collaboration avec l'industrie pour développer une solution pratique pour une authentification renforcée sur le Web », a déclaré Brett McDowell, directeur exécutif de l'Alliance FIDO. "Avec cette étape importante, nous entrons dans une nouvelle ère de protection de l'authentification FIDO omniprésent et soutenu par matériel pour tous ceux qui utilisent Internet". L'annonce par WebAuthn du W3C et de l'Alliance FIDO est, nous l'espérons, un pas vers la réalisation connexions sans mot de passe dans sites Internet. Nous nous attendons à voir de nombreux services Web implémenter WebAuthn dans les mois à venir et, finalement une utilisation plus large sur le Web dans son ensemble.
